# parkSmart-parking reservation platform
A real-time parking reservation and monitoring system for efficeint slot management.


## Project Overview
ParkSmart is an integrated solution designed to streamline parking management. It allows users to reserve slots based on vehicle type, tracks occupancy in real-time, and automates fee calculations and overstay penalties.
